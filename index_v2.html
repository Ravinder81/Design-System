Overview

Design Principles

Four core principles guide every decision in this design system.

🎯

Clarity First

Every component communicates its purpose instantly. Reduce cognitive load through consistent patterns, clear hierarchy, and predictable interactions.

⚖️

Consistency

Same patterns, tokens, and components across the entire product. Consistency builds user trust and reduces learning time across teams.

♿

Accessibility

WCAG 2.1 AA compliant. All interactive elements have focus states, ARIA labels, and meet minimum contrast ratios of 4.5:1.

📦

Scalability

Built with design tokens and component variants that scale from single-page apps to complex enterprise platforms without redesign.

Theme Roller Quick Start

Apply this design system to your APEX app in 3 steps using Theme Roller.

1
Open Theme Roller
Shared Components → Themes → Universal Theme → Theme Roller. Set --ut-palette-primary to #007FA3
2
Add CSS Tokens to app.css
Copy the :root token block from this design system into your APEX application's global CSS (Shared Components → Static Application Files → app.css)
3
Use Component CSS Classes
Apply .btn-primary, .kpi-card, .badge-success etc. via the HTML Expression column in IR or custom region templates in your APEX pages.
